# OAuth Branding Pages

These files are for Google OAuth branding verification:

- `index.html` (homepage)
- `privacy.html` (privacy policy)
- `terms.html` (terms of service)

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open repo settings: `Settings -> Pages`.
3. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
4. Save and wait for Pages to publish.

## Expected URLs (for this repo)

- Homepage: `https://dieselmikek.github.io/InvoiceExtractor/`
- Privacy Policy: `https://dieselmikek.github.io/InvoiceExtractor/privacy.html`
- Terms of Service: `https://dieselmikek.github.io/InvoiceExtractor/terms.html`

Use these URLs in Google Cloud `Google Auth Platform -> Branding`.
